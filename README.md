## ContractWiser Analytical Engine

### Github

git clone git@github.com:ronrlin/cw-engine.git cw-engine/

### Installing AgreementCorpus and data

cd cw-engine
python config.py

### Installing Flask & Libraries

pip install -r requirements.txt -t lib
